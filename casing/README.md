# 3D printed casing for clock_box

The casing consists of a top and bottom shell which fits tightly around the clock_box PCB.

To hold the shells together, the following hardware is needed:

  * 2x M2 Hex Socket screws of length 6 mm
  * 2x M2 Nuts of thickness 1.6 mm


It was modeled in [Openscad](https://openscad.org/) version `2024.06.04.ai19515`.

The dimensions have been optimized to print well on a Bambulab A1 using PLA filament.
